
# Haleta

It is a mobile app that shows the nearest ambulance and other features.


## Features

- Nearest ambulance
- Google map direction
- Health news



## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd haleta
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  cd client
  npm expo start -c
```


## Tech Stack

**Client:** React Native, Native Wind
